🐍 Easy DSA Problems

Master Python Lists & Dictionaries with these beginner-friendly challenges

1️⃣ Count Occurrences (Easy) | ⏱ ~5 min

Task: Build a frequency dictionary from a list of words. Do not use collections.Counter; implement the counting logic manually.

words = ["apple", "banana", "apple", "orange", "banana", "apple"]

➡️ Expected: {"apple": 3, "banana": 2, "orange": 1}

Edge case: Empty list → {}

2️⃣ Invert a Dictionary (Easy → Medium) | ⏱ ~10 min

Task: Flip a dictionary so values become keys, and keys with the same value are grouped in a list.

original = {"a": 1, "b": 2, "c": 1, "d": 3}

➡️ Expected: {1: ["a", "c"], 2: ["b"], 3: ["d"]}

Consideration: Inversion fails if values are unhashable (e.g., lists).

3️⃣ Merge Two Lists → Dict (Easy) | ⏱ ~3 min

Task: Pair two equal‑length lists (keys and values) into a dictionary.

keys   = ["name", "age", "city"]
values = ["Zeeshan", 22, "Delhi"]

➡️ Expected: {"name": "Zeeshan", "age": 22, "city": "Delhi"}

Bonus: Handle unequal lengths by padding the shorter list with None.

💡 Tip: These problems reinforce dictionary operations, loops, and data organization — crucial for interview success!
