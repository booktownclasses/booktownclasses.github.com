The range() Function

for i in range(5):       # 0 1 2 3 4

🧑‍💻 Exercise: Sum of First N Numbers

Ask the user for a number n. Calculate and print the sum of all numbers from 1 to n using a for loop.

day1_exercise_solution.py
n = int(input("Enter a number: "))
total = 0
for i in range(1, n + 1):
    total += i
print(f"Sum of 1 to {n} is {total}")
Enter a number: 10
Sum of 1 to 10 is 55
💡 Instructor Tip: The loop variable i takes each value from range() one at a time. range(1, n+1) is used because range excludes the stop value.

Day 2 While Loops & Indefinite Iteration

Learning Objectives: Understand when to use while vs for. Write condition-based loops. Use sentinel values and counters. Avoid infinite loops.

While Loop Basics

day2_while_basic.py
count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1
🤔 for vs while — Decision Guide Infographic

🔁 for loop

  • Known number of iterations
  • Iterating over sequences
  • Example: print 1–10

🔄 while loop

  • Unknown number of iterations
  • Waiting for a condition
  • Example: input validation

Sentinel-Controlled Loop

day2_sentinel.py
text = ""
while text != "quit":
    text = input("Enter text (type 'quit' to stop): ")
    if text != "quit":
        print(f"You entered: {text}")
print("Goodbye!")

🧑‍💻 Exercise: Countdown Timer

Ask the user for a starting number and print a countdown from that number to 1, then print "Blast off!" 🚀

day2_exercise_solution.py
start = int(input("Enter countdown start: "))
while start > 0:
    print(start)
    start -= 1
print("🚀 Blast off!")
Enter countdown start: 5
5 4 3 2 1 🚀 Blast off!
⚠️ Infinite Loop Alert: Always ensure the loop condition will eventually become False. Forgetting to update the counter is the most common cause. Show Ctrl+C to stop.

Day 4 Break, Continue & Loop-else

Learning Objectives: Exit loops early with break. Skip iterations with continue. Use the else clause with loops. Combine these with nested loops.

🛑 Loop Control Commands Reference
🛑

break

Exit the loop immediately

⏭️

continue

Skip to next iteration

🏁

loop-else

Runs if no break occurred

break Example

day4_break.py
for num in range(1, 200):
    if num % 7 == 0 and num % 5 == 0:
        print(f"Found: {num}")   # 35
        break

continue Example

day4_continue.py
for num in range(1, 11):
    if num % 2 == 0:
        continue
    print(num)   # 1 3 5 7 9

Loop-else (Prime Checker)

day4_loop_else.py
num = 29
for i in range(2, int(num**0.5)+1):
    if num % i == 0:
        print("Not prime"); break
else:
    print("Prime")
⚠️ Note: Loop-else is unique to Python. It runs only if the loop finishes normally (no break), not if the loop didn't run at all.

Day 5 Mini-Project: Number Guessing Game

Learning Objectives: Combine loops, conditionals, and user input into a complete game. Use random module. Give feedback. Track attempts.

Project Requirements

  1. Generate a random number between 1 and 100.
  2. Let the user guess until they get it right.
  3. After each guess, say "Too high!" or "Too low!".
  4. When correct, print the number of attempts.
  5. Bonus: Limit the user to 10 attempts.
guessing_game.py
import random
secret = random.randint(1, 100)
attempts = 0
while True:
    guess = int(input("Your guess: "))
    attempts += 1
    if guess < secret: print("Too low!")
    elif guess > secret: print("Too high!")
    else: print(f"Correct! {attempts} attempts"); break
🚀 Extension: Add difficulty levels, high score tracking, or hints when within 5.

Week 2 Practice Problems

Test your understanding with these problems covering all Week 2 concepts.

🔰 Beginner Level

1. Factorial Calculator

Problem: Ask for a number n and compute n! (factorial) using a for loop. Example: 5! = 5 × 4 × 3 × 2 × 1 = 120.

n = int(input("Enter a number: "))
factorial = 1
for i in range(1, n + 1):
    factorial *= i
print(f"{n}! = {factorial}")
2. Sum of Digits

Problem: Ask for an integer and calculate the sum of its digits using a while loop. Example: 1234 → 1+2+3+4 = 10.

num = int(input("Enter an integer: "))
total = 0
temp = abs(num)
while temp > 0:
    total += temp % 10
    temp //= 10
print(f"Sum of digits = {total}")
3. Hollow Square Pattern

Problem: Print a hollow square of * with side length n using nested loops.

n = 5
for i in range(n):
    for j in range(n):
        if i == 0 or i == n-1 or j == 0 or j == n-1:
            print("*", end=" ")
        else:
            print(" ", end=" ")
    print()

⚡ Intermediate Level

4. Fibonacci Series

Problem: Print the first n Fibonacci numbers using a for loop. The series starts 0, 1, 1, 2, 3, 5, 8, 13, ...

n = int(input("How many Fibonacci numbers? "))
a, b = 0, 1
for _ in range(n):
    print(a, end=" ")
    a, b = b, a + b
5. FizzBuzz (Classic Interview Question)

Problem: Print numbers 1 to 50. For multiples of 3 print "Fizz", for multiples of 5 print "Buzz", for multiples of both print "FizzBuzz".

for num in range(1, 51):
    if num % 3 == 0 and num % 5 == 0:
        print("FizzBuzz")
    elif num % 3 == 0:
        print("Fizz")
    elif num % 5 == 0:
        print("Buzz")
    else:
        print(num)
