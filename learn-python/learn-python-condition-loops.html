Condition & Loops

Control program flow: Make decisions with if and repeat code with loops.

🔄

For Loop

Iterates over a sequence (list, string, range)

🔁

While Loop

Repeats as long as a condition is true

⚙️

range()

Generates a sequence of numbers

⛔

break / continue

break exits, continue skips

If‑Elif‑Else Statements

age = 18
if age < 13:
    print("You are a child")
elif age < 18:
    print("You are a teenager")
else:
    print("You are an adult")
Output: You are an adult

For Loops

Use a for loop to repeat for each item in a sequence.

Beginner Examples

# Loop through a list
fruits = ["apple", "banana", "orange"]
for fruit in fruits: print(fruit)

# Loop through a string
for letter in "Python": print(letter)

# range(stop)
for i in range(5): print(i)
apple banana orange P y t h o n 0 1 2 3 4

Intermediate Examples

# enumerate
for i, fruit in enumerate(fruits): print(f"{i}: {fruit}")

# nested loop
for i in range(1,4):
    for j in range(1,4): print(f"{i}x{j}={i*j}")

# else with loop
for x in range(3): print(x)
else: print("Loop ended without break")
0: apple 1: banana 2: cherry 1x1=1 ... 3x3=9 0 1 2 Loop ended without break

range() – The Loop Counter

range(start, stop, step) generates a sequence of numbers.

1️⃣

range(stop)

range(5) → 0,1,2,3,4

2️⃣

range(start,stop)

range(2,6) → 2,3,4,5

3️⃣

range(start,stop,step)

range(1,10,2) → 1,3,5,7,9

Three flavours in action

# Case 1: range(stop)
for i in range(4): print(i)
# Case 2: range(start,stop)
for i in range(10,13): print(i)
# Case 3: range(start,stop,step)
for i in range(2,11,2): print(i)
# negative step (countdown)
for i in range(5,0,-1): print(i)
Case 1: 0 1 2 3 Case 2: 10 11 12 Case 3: 2 4 6 8 10 Countdown: 5 4 3 2 1

While Loops

A while loop repeats as long as a condition is True.

Beginner Examples

count = 0
while count < 5:
    print(count)
    count += 1

# guess the number
secret = 7
guess = 0
while guess != secret:
    guess = int(input("Guess 1-10: "))
print("You got it!")
0 1 2 3 4 Guess 1-10: 3 ... 7 You got it!

Intermediate Examples

# break
i = 1
while True:
    if i > 5: break
    print(i); i+=1

# continue (skip even)
num = 0
while num < 10:
    num+=1
    if num%2==0: continue
    print(num)

# while with else
x=1
while x<=3: print(x); x+=1
else: print("Loop finished")
1 2 3 4 5 1 3 5 7 9 1 2 3 Loop finished
