If‑Elif‑Else Statements

The if statement lets you execute code only when a condition is True. Use elif for additional checks and else for a default block.

conditional.py
age = 18
if age < 13:
    print("You are a child")
elif age < 18:
    print("You are a teenager")
else:
    print("You are an adult")
You are an adult

Note: • if must always come first. • elif and else are optional. • Indentation defines the code block.

For Loop

A for loop repeats a block of code for each item in a sequence. It works with strings, lists, tuples, dictionaries, and the range() function.

Syntax: for variable in iterable: ...

Beginner Example

Loop through a list, a string, and a simple range(stop).

for_loop_basic.py
# Loop through a list
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# Loop through a string
for letter in "Python":
    print(letter)

# range(stop)
for i in range(5):
    print(i)
apple banana orange P y t h o n 0 1 2 3 4

Intermediate Example

Using enumerate() to get index, nested loops, and else with a loop.

for_loop_intermediate.py
# enumerate
fruits = ["apple", "banana", "cherry"]
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

# Nested loop (multiplication table)
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")

# Loop with else (runs if no break)
for x in range(3):
    print(x)
else:
    print("Loop ended without break")

range() – The Loop Counter

The range() function generates a sequence of numbers without storing them all in memory. It's the most common partner of the for loop.

Syntax: range(start, stop, step) • start (default 0) – first number. • stop (required) – up to but not including this number. • step (default 1) – difference between each number.

1️⃣

range(stop)

range(5) → 0,1,2,3,4

2️⃣

range(start, stop)

range(2,6) → 2,3,4,5

3️⃣

range(start, stop, step)

range(1,10,2) → 1,3,5,7,9

Beginner Example (stop only)

range_stop.py
# Print numbers 0 to 3
for i in range(4):
    print(i)
0
1
2
3

Intermediate Example (start, stop, step)

range_all.py
# Case 1: range(stop)
print("Case 1:")
for i in range(4): print(i)

# Case 2: range(start, stop)
print("Case 2:")
for i in range(10, 13): print(i)

# Case 3: range(start, stop, step) – even numbers
print("Case 3:")
for i in range(2, 11, 2): print(i)

# Negative step (countdown)
print("Countdown:")
for i in range(5, 0, -1): print(i)

Case 3:

While Loop

A while loop keeps repeating as long as a condition remains True. Be cautious – if the condition never becomes False, you get an infinite loop.

Syntax: while condition: ...

Beginner Example

Simple counter and a guessing game.

while_basic.py
# Count from 0 to 4
count = 0
while count < 5:
    print(count)
    count += 1

# Guess the number
secret = 7
guess = 0
while guess != secret:
    guess = int(input("Guess the number (1-10): "))
print("You got it!")

Intermediate Example

Using break, continue, and while-else.

while_intermediate.py
# break – stop early
i = 1
while True:
    if i > 5: break
    print(i)
    i += 1

# continue – skip even numbers
num = 0
while num < 10:
    num += 1
    if num % 2 == 0: continue
    print(num)

# while with else
x = 1
while x <= 3:
    print(x)
    x += 1
else:
    print("Loop finished")

Practice Problems

Strengthen your understanding with these solved DSA‑style problems. Each uses conditions, loops, or both.

🔰 Beginner Level

1. Check If a Number is Prime

Problem: Write a program that takes an integer and prints whether it is prime or not.

num = int(input("Enter a number: "))
is_prime = True
if num > 1:
    for i in range(2, int(num**0.5)+1):
        if num % i == 0:
            is_prime = False
            break
    print("Prime" if is_prime else "Not Prime")
else:
    print("Not Prime")
2. Sum of First n Natural Numbers

Problem: Ask for a positive integer n and print the sum 1+2+...+n using a loop.

n = int(input("Enter n: "))
total = 0
for i in range(1, n+1):
    total += i
print("Sum =", total)
3. Reverse a String (without slicing)

Problem: Input a string and print its reverse character by character using a loop.

text = input("Enter a string: ")
rev = ""
for ch in text:
    rev = ch + rev
print("Reversed:", rev)

⚡ Intermediate Level

4. Fibonacci Series up to n Terms

Problem: Print the first n terms of the Fibonacci series using a loop.

n = int(input("How many terms? "))
a, b = 0, 1
for _ in range(n):
    print(a, end=' ')
    a, b = b, a + b
5. Count Digits, Letters, and Spaces

Problem: Read a sentence and count digits, alphabets, and spaces using a loop.

sentence = input("Enter a sentence: ")
digits = letters = spaces = 0
for ch in sentence:
    if ch.isdigit(): digits += 1
    elif ch.isalpha(): letters += 1
    elif ch == ' ': spaces += 1
print(f"Digits: {digits}, Letters: {letters}, Spaces: {spaces}")
6. Print the Multiplication Table from 1 to 10 (Cleanly)

Problem: Use nested loops to print a beautifully aligned multiplication table from 1 to 10.

for i in range(1, 11):
    for j in range(1, 11):
        print(f"{i*j:4}", end="")
    print()
