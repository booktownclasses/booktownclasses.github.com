List, Tuples, Set

Collection data types: Store multiple items with different rules. Each type is optimized for a specific use case.

📋

List

Mutable, ordered, allows duplicates

📦

Tuple

Immutable, ordered, allows duplicates

🔢

Set

Mutable, unordered, no duplicates

Lists – Mutable Ordered Collection

A list is the most versatile Python data structure. It can hold items of different types and you can change it after creation.

Creating & Accessing

list_basics.py
fruits = ["apple", "banana", "cherry"]
print(fruits[0])      # first element
print(fruits[-1])     # last element
print(fruits[1:3])    # slicing ['banana', 'cherry']
apple cherry ['banana', 'cherry']

Common Operations

list_operations.py
nums = [2,5,1,8,3]

# Add / Remove
nums.append(9)        # add to end → [2,5,1,8,3,9]
nums.insert(1, 4)     # insert at index 1 → [2,4,5,1,8,3,9]
nums.remove(1)        # remove first occurrence of 1
popped = nums.pop(2)  # remove & return item at index 2

# Sorting & Searching
nums.sort()           # in-place sort
nums.reverse()        # reverse order
idx = nums.index(5)   # find index of 5 (first)
cnt = nums.count(4)   # count occurrences of 4

# Copy
copy_list = nums.copy()  # shallow copy
print(nums)
print("Pop:", popped, "| Index of 5:", idx)
[2,3,4,5,8,9] Pop: 1 | Index of 5: 3

List Comprehension (Quick creation)

list_comprehension.py
squares = [x**2 for x in range(1,6)]  # [1,4,9,16,25]
evens = [x for x in range(10) if x%2==0]
print(squares)
print(evens)
[1, 4, 9, 16, 25] [0, 2, 4, 6, 8]

Tuples – Immutable Ordered Collection

A tuple is like a list, but cannot be modified after creation. Use it for fixed data (coordinates, days of week).

Creating & Accessing

tuple_basics.py
colors = ("red", "green", "blue")
print(colors[0])      # red
print(colors[-1])     # blue
# Tuples support slicing just like lists
print(colors[:2])     # ('red', 'green')
red blue ('red', 'green')

Operations & Immutability

tuple_ops.py
nums = (5,2,5,8,5)
print(len(nums))       # length → 5
print(nums.count(5))   # count → 3
print(nums.index(8))   # index of 8 → 3

# Unpacking
a, b, c = (1,2,3)
print(a,b,c)

# Cannot modify → error
# nums[0] = 10   ❌ TypeError
5 3 3 1 2 3

Sets – Mutable Unordered Unique Collection

A set stores distinct values with no duplicates. Ideal for membership tests, removing duplicates, and mathematical operations.

Creating & Basic Operations

set_basics.py
nums = {1,2,3,3,4,5,5}   # duplicates removed automatically
print(nums)               # {1,2,3,4,5}

nums.add(6)               # add element
nums.remove(1)            # remove (error if not present)
nums.discard(10)          # removes if exists, no error
popped = nums.pop()       # removes a random element

print(len(nums), nums)
{1, 2, 3, 4, 5} 4 {2, 3, 4, 5, 6}

Set Operations (Math-like)

set_operations.py
a = {1,2,3,4}
b = {3,4,5,6}

print(a.union(b))               # {1,2,3,4,5,6}
print(a.intersection(b))        # {3,4}
print(a.difference(b))          # {1,2}
print(b.difference(a))          # {5,6}
print(a.symmetric_difference(b))# {1,2,5,6}
{1,2,3,4,5,6} {3,4} {1,2} {5,6} {1,2,5,6}

Practice Problems (Solved)

🔰 Beginner Level

1. Find the Maximum Element in a List

Problem: Given a list of numbers, find and print the largest number without using max().

nums = [4, 12, 7, 19, 3, 5]
max_val = nums[0]
for n in nums:
    if n > max_val:
        max_val = n
print("Maximum:", max_val)
2. Remove Duplicates from a List (Preserve Order)

Problem: Remove duplicate values while keeping the original order intact.

items = [1,2,3,2,4,5,1]
seen = []
for x in items:
    if x not in seen:
        seen.append(x)
print(seen)
3. Count Specific Element in a Tuple

Problem: Count how many times a given value appears in a tuple (without .count()).

data = (5,3,8,5,9,5,2)
target = 5
count = 0
for x in data:
    if x == target:
        count += 1
print(f"{target} appears {count} times")

⚡ Intermediate Level

4. Merge Two Sorted Lists into One Sorted List

Problem: Given two sorted lists, create a new sorted list containing all elements (without sort()).

a = [1,4,6]
b = [2,3,5]
merged = []
i = j = 0
while i < len(a) and j < len(b):
    if a[i] < b[j]:
        merged.append(a[i]); i+=1
    else:
        merged.append(b[j]); j+=1
merged.extend(a[i:])
merged.extend(b[j:])
print(merged)
5. Find Common Elements in Two Sets

Problem: Using sets, find and display the intersection of two sets.

set1 = {"apple", "banana", "cherry"}
set2 = {"banana", "date", "cherry"}
common = set1.intersection(set2)
print("Common:", common)
6. Convert List of Tuples to Dictionary

Problem: Given a list of (key, value) tuples, create a dictionary for fast lookups.

pairs = [("name","Aarav"), ("grade",8), ("marks",84.5)]
student = dict(pairs)
print(student)
print(student["name"])
